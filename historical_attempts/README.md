# Historical Attempts

This folder stores experimental scripts that were tried during development but are not part of the final submission route.

The final working pipeline is kept in the repository root:

```text
dataset.py
submission_utils.py
train_dinov2_lora.py
```

Historical scripts include frozen-feature ensembles, INR descriptors, label propagation, nearest-neighbor analysis, SAM preprocessing, and meta-INR experiments. They are kept for reproducibility and presentation discussion, but the current best public score comes from DINOv2 LoRA/PEFT fine-tuning.
